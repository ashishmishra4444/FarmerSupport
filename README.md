# Farmer Support Platform

Full-stack MERN starter for a farmer support and market price platform.


## Setup

### Backend
1. Install dependencies with `npm install`
2. Start the API with `npm run dev`

### Frontend

1. Install dependencies in `client` with `npm install`
2. Start the app with `npm run dev`

## Suggested Demo Flow

1. Register a farmer account
2. Add a few products from the farmer dashboard
3. Register a buyer account
4. Place an order from the buyer dashboard
5. Mark payment as verified through the backend payment endpoint
6. Register or seed an admin account and review the admin dashboard charts
